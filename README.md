# Firmable Assignment Theme

A custom WordPress theme built for the Firmable Assignment.  
This theme demonstrates palette‑driven colors, reusable block patterns, responsive navigation, and reviewer‑friendly code organization.

---

## ✨ Features
- Palette‑driven design via `theme.json`
- Reusable block patterns for quick content creation
- Responsive navigation with `navigation.js`
- Consistent button, header, and footer styling
- Accessibility polish with hover animations and adaptive icon colors
